How it works

Five-stage pipeline

Every video frame passes through detection, tracking, owner-linking, alert evaluation, and annotation before hitting the dashboard.

01
YOLO Detection
YOLOv8 runs on a downscaled copy of each keyframe, finding persons and bags.
yolov8n.pt
02
DeepSORT Tracking
Assigns persistent integer IDs across frames using appearance embeddings.
MobileNet
03
Owner Linking
Each bag is assigned the nearest person at first detection. Updated each frame.
proximity
04
Alert Engine
Timer starts when owner moves away. Escalates Safe → Warning → Critical.
configurable
05
Dashboard
Streamlit live feed with metrics, event log, screenshots, and performance stats.
Streamlit
Alert levels
●   SAFE  ·  Owner within 150 px  ·  No timer
▲   WARNING  ·  Unattended > 5 s  ·  Alarm at 50% volume
■   CRITICAL  ·  Unattended > 10 s  ·  Alarm + screenshot saved
Features

What's included

▶
Real-time detection
Frame-skip optimisation runs YOLO every N frames while DeepSORT updates every frame for smooth tracking.
⚙
Config file
All parameters in config.yaml — model, thresholds, logging, UI. No code changes needed.
◎
Structured logging
Coloured console + rotating .log + one-JSON-per-line .jsonl for downstream parsing.
■
Screenshot gallery
CRITICAL alerts auto-capture annotated JPEG screenshots. Manual snap button in the dashboard.
◈
Owner-linking
Nearest-person assignment at first detection. Dashed line drawn between bag and owner in the feed.
◑
Performance panel
Live FPS counter, avg YOLO inference ms, and frame counter visible in the Streamlit sidebar.
Install

Quick start

bash
# 1. Clone the repo git clone https://github.com/your-username/smart-unattended-detection.git cd smart-unattended-detection # 2. Install dependencies pip install -r requirements.txt # 3. Launch the dashboard (YOLOv8 weights auto-download on first run) streamlit run app.py # — or — headless mode on a video file python main.py --source demo/airport.mp4 --no-display
Config

config.yaml

Every tunable parameter lives here. Edit and restart — no Python changes required.

yaml
# Model — swap yolov8n → yolov8s/m/l for higher accuracy model: path: "yolov8n.pt" confidence_threshold: 0.45 device: "cpu" # or "cuda" / "mps" # Alert timers alerts: warning_time: 5 # seconds before WARNING critical_time: 10 # seconds before CRITICAL # Performance — skip frames to reduce CPU load performance: process_every_n_frames: 2 # Logging output: log_level: "INFO" log_to_file: true
